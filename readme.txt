== Description ==
Автоматизируйте и защитите переводы ваших клиентов с помощью плагина для безопасных сделок от эквайрингового сервиса Тинькофф Кассы.
= Для кого =
* Интернет-магазины
* Биржи услуг
* Онлайн-школы
* Сервисы продажи билетов, бронирования жилья и гостиниц
= Как это работает =
Клиент оплачивает заказ, а нужная сумма списывается с его карты и хранится на промежуточном счете. Продавец/поставщик видит уведомление об оплате и отправляет товар/оказывает услугу. После получения заказа клиент отправляет подтверждение, а оплаченная сумма, за вычетом комиссии площадки, поступает на счет продавца.

== Installation ==
1. Загрузите архив с плагином через меню "Плагины – Добавить новый" или разархивируйте плагин в папку "/wp-content/plugins/".
2. Активируйте плагин через меню "Плагины" вашего сайта.
3. Перейдите в меню "Woocommerce – Настройки – Платежи", включите способ оплаты Tinkoff Secure deal, перейдите в раздел "Управление" и заполните настройки плагина. Сохраните изменения.
4. Проведите тестовые платежи и переключитесь на боевой режим после тестирования.
= Как получить ключи для настройки =
Чтобы получить необходимые тестовые и боевые доступы для настройки плагина, заполните форму на сайте https://www.tinkoff.ru/business/open-api/. После этого с вами свяжется менеджер Тинькофф и вышлет требуемую информацию на email, указанный в заявке.
